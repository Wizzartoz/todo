# todo
App Features:
    Users can create new account.
    Users can create/check/delete new tasks.
    Users can sort, serarch, filter, tasks.

Technologies Used:
Backend

    Java 11
    Spring Boot
    Spring Security
    Spring Data
    Spring REST
    PostgreSQL
    Maven

Frontend

    Angular
    Angular Material UI
    Typescript

# How to run it

  1. clone repository
  2. enter you data in .env
  3. execute command docker-compose up -d --build
